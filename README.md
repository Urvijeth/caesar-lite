#  caesar-lite

**caesar-lite** is a simple and aesthetic Python tool that performs **Caesar Cipher encryption and decryption** on text and files.  
It includes both a **Tkinter-based dark GUI** and a **command-line interface (CLI)** for flexibility and learning purposes.

---

##  Features
- Encrypt & decrypt **text** or **files**
- Clean, modern **dark GUI**
- Simple **CLI support**
- Works completely offline
