#  caesar-lite

**caesar-lite** is a simple and aesthetic Python tool that performs **Caesar Cipher encryption and decryption** on text and files.  
It includes both a **Tkinter-based dark GUI** and a **command-line interface (CLI)** for flexibility and learning purposes.

---

##  Features
- Encrypt & decrypt **text** or **files**
- Clean, modern **dark GUI**
- Simple **CLI support**
- Works completely offline

## Installation

1. Clone the repository:

```bash
   git clone https://github.com/Urvijeth/caesar-lite.git   
```
2. Navigate to the project directory:

```bash
cd caesar-lite
```
3. Install the required :

```bash
sudo dnf install python3-tkinter   # Fedora
sudo apt install python3-tk     # Ubuntu / Debian / Kali

```
3. Run the Program :

```bash
python -m src.main

```
